## Prerequisites

- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)

## Get project

```bash
  git clone https://github.com/meherfadhleoui/test-technique

  cd test-technique
```

## Build and Run the Project

```bash
docker-compose up
```

## visit : http://localhost:4200

## To stop the Project

```bash
docker-compose down
```
